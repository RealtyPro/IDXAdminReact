# RealtiPro - Smart Interaction Logger

A modern web application for capturing, storing, and analyzing WhatsApp & call interactions with ease.

## Features

- Real-time chat logging
- Call recording playback
- Seamless WhatsApp integration
- Advanced analytics tools

## Getting Started

### Prerequisites

- Node.js 18.x or later
- npm 9.x or later

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/realtipro.git
cd realtipro
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
